# 백준 1212 : 8진수 2진수

#### Date : 21/08/21

#### Lang : Python

#### Question : [백준 1212](https://www.acmicpc.net/problem/1212)

## 최종 답안 

```python
oct = bin(int(input(), 8))
print(oct[2:])

```

이전에 풀었던 2진수 8진수 문제와 비슷하다.  
이번엔 굳이 계산을 해주지 않고 내장 함수를 사용해 주었다.   
8진수로 값을 입력받은 뒤, 그걸 2진수로 변환하여 oct 변수에 넣어주었다.  
출력할때는 [2:]로 슬라이싱 해서 출력해주었는데,  
8진수임을 나타내는 0o를 빼주기 위함이다.

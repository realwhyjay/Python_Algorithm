# 백준 11726 : 2xn 타일링

#### Date : 21/09/14

#### Lang : Python

#### Question : [백준 11726](https://www.acmicpc.net/problem/11726)

## 최종 답안 

```python
n = int(input())
result = [0, 1, 2]
for i in range(3, 1001):
    result.append(result[i-2]+result[i-1])
    
print(result[n] % 10007)

```

점화식을 구해보면  n = (n-2) + (n-1) 과 같다.

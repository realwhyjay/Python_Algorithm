#### Lang : Python

#### Question : [백준 2193](https://www.acmicpc.net/problem/2193)

## 최종 답안 

```python

# N   이친수
# 1 -> 1
# 2 -> 10
# 3 -> 100 101
# 4 -> 1000 1001 1010
# 5 -> 10000 10001 10010 10100 10101
# 6 -> 100000 100001 101001 100101 100010 100100 101000 101010

n = int(input())

pinary = [0, 1, 1, 2]

for i in range(4, 91):
    pinary.append(pinary[i-1]+pinary[i-2])

print(pinary[n])
```

주석에 달아둔 것처럼,  
N의 개수에 따라 나올 수 있는 이친수의 개수를 구해서 그 개수 사이의 규칙을 찾아주었다.

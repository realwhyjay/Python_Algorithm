## 최종 답안 

```python
n, b = input().split()
n = n[::-1]
b = int(b)

num = '0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ'
result = 0

for i in range(len(n)):
    result += num.index(n[i])*(b**i)

print(result)

```

입력받은 수를 10진수로 바꾸어주는 문제.  
string 값으로 수들의 목록을 만들어 준뒤,  인덱스를 활용하여 해당 문자가 어떤 값을 가지는지를 나타내주었다.

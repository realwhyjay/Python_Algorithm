## 1차 답안 

```python
n = int(input())
stack = []
for _ in range(n):
    command = list(input().split())
    if command[0] == 'push':
        stack.append(command[1])
    elif command[0] == 'pop':
        if not stack:
            print(-1)
        else:
            print(stack.pop())
    elif command[0] == 'top':
        if not stack:
            print(-1)
        else:
            print(stack[-1])
    elif command[0] == 'size':
        print(len(stack))
    else:
        print(1 if not stack else 0)

```

구현도 어렵지 않았다. 쉬운 문제 골라푸는 양아치 된 기분~ ㅎ  
IDE에서는 아무런 문제가 없지만, 백준에서는 시간초과가 났다.  
간단하게 아래의 코드를 추가해서 시간을 줄였다.  

계속 하다보니까 readline으로 시간초과를 해결 할 수 있는 경우가 많아보이는데,  
이제는 그냥 애초에 아래 코드를 디폴트로 박아둬야겠다.

```python
import sys
input=sys.stdin.readline
```

## 최종 답안 

```python
str = list(input())
result = ""
for i in range(len(str)):
    if str[i] == ' ':
        result += ' '
    elif ord('a') <= ord(str[i]) <= ord('z'):
        temp = ord(str[i])+13
        if temp > ord('z'):
            temp = (temp % ord('z'))+ord('a')-1
        result += chr(temp)
    elif ord('A') <= ord(str[i]) <= ord('Z'):
        temp = ord(str[i])+13
        if temp > ord('Z'):
            temp = (temp % ord('Z'))+ord('A')-1
        result += chr(temp)
    else:
        result += str[i]

print(result)

```

알 수 없는 이유로 시간이 오래걸렸는데, 원래는 elif 문의 조건을  
`ord('a') <= ord(str[i]) <= ord('z')` 가 아닌 `srt[i].islower`로 주었다.  
그런데 이렇게 islower 함수를 사용하는 경우에, 대문자들도 조건에 걸리는 이상한 일이 발생.  
따로 함수를  체크해봐도 문제 없었는데 . 저 for문 안에 들어가는 순간에만 이상해진다.  
어찌됐든 조건을 수정해서 문제를 해결했다.

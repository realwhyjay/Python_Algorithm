#### Question : [백준 5585](https://www.acmicpc.net/problem/5585)

## 최종 답안 

```python
pay = int(input())
money = 1000-pay
count = 0
change = money
kind = [500, 100, 50, 10, 5, 1]

for i in kind:
    if change == 0:
        break
    else:
        count += int(change/i)
        change = int(change % i)

print(count)

```

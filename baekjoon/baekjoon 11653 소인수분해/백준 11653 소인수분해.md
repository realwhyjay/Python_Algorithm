# 백준 11653 : 소인수분해

#### Date : 21/09/05

#### Lang : Python

#### Question : [백준 11653](https://www.acmicpc.net/problem/11653)

## 최종 답안 

```python
num = int(input())

check = 2

while (num != 1):
    if num % check == 0:
        print(check)
        num /= check
    else:
        check += 1

```

입력받은 수를 소인수분해하여, 그 값들을 출력해주는 문제이다.  
입력받은 수를 2부터 시작하는 check 값을 이용해 나누어주고, check을 1씩 늘려주는식으로 체크했다.  
문제를 풀긴 했지만 솔루션에 문제가 하나 있는데,   
만일 check 값이 2일 경우에 num 값을 check값으로 나눈 나머지가 0이라면   
이후에 check 값이 4가 되었을때도 똑같이 나머지가 0이 될 것이 분명한데 굳이 또 그걸 체크해주고 있다는 점이다.  
효율을 위해서라면 에라토스테네스의 체를 활용하거나 하면 될 것 같았는데,  
해당 문제의 경우에는 시간 제한이 그렇게 빡빡하지 않아서 이정도만으로도 충분히 문제 해결이 가능했다.

# 백준 6236 : 용돈 관리

#### Date : 21/03/21

#### Lang : Python

#### Question : [백준 6236](https://www.acmicpc.net/problem/6236)

## 최종 답안 

```python
n, m = map(int, input().split())
money = []

for _ in range(n):
    money.append(int(input()))

start = max(money)  # 시작점 (최소값)
end = sum(money)		# 끝점 (최대값)
result = sum(money)

while start <= end:		# 이분 탐색 시작
    mid = (start+end)//2		# mid = 현재 시도하는 k 값
    count = 0		# 현재 탐색에서의 인출 횟수 count
    sum = 0		# 현재 탐색에서의 부분 합

    for i in money: # 리스트를 돌면서 탐색을 시작한다
        sum += i		# mid 값을 초과하면 횟수를 +1 해준다
        if sum > mid:
            count += 1
            sum = i		# 이거 왜 해주는거지 스벌

    if sum > 0:
        count += 1		# 마지막에 남으면 횟수 1 추가해준다 (무슨말이지 스벌)

    if count > m:		# 인출 횟수가 M보다 크면 안된다
        start = mid+1		# 탐색 범위를 뒤쪽으로 이동시킨다
    else:			# 인출 횟수가 M보다 작거나 같은 경우, 조건은 만족한다
        if mid < result:
            result = mid		# result 값보다 현재의 k 값이 더 작다면 업데이트 해준다
        end = mid-1

print(result)

```

# 백준 1182 : 부분수열의 합

#### Date : 21/04/15

#### Lang : Python

#### Question : [백준 1182](https://www.acmicpc.net/problem/1182)

## 최종 답안

```javascript
from itertools import combinations

n, s = map(int, input().split())
lis = list(map(int, input().split()))
count = 0

for i in range(1, n+1):
    comb = list(combinations(lis, i))
    for j in comb:
        if sum(j) == s:
            count += 1

print(count)
```

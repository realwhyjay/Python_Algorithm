# 백준 2577 : 숫자의 개수

#### Date : 21/05/16

#### Lang : Python

#### Question : [백준 2577](https://www.acmicpc.net/problem/2577)

## 최종 답안 

```python
abc = []
for _ in range(3):
    abc.append(int(input()))

ans = list(str(abc[0]*abc[1]*abc[2]))

for i in range(10):
    print(ans.count(str(i)))

```

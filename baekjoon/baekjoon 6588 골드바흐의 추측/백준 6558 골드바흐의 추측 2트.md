# 백준 6558 : 골드바흐의 추측 2트

#### Lang : Python

#### Question : [백준 6558](https://www.acmicpc.net/problem/6558)

## 최종 답안 

```python
import sys
input = sys.stdin.readline
N = 1000000
# 에라토스테네스 체
check = [False, False]+[True]*(N-1)
for i in range(2, int((N+1)**0.5)+1):
    if check[i] == True:
        for j in range(2*i, N+1, i):
            check[j] = False

# 골드바흐 파티션 출력
while True:
    n = int(input())
    if n == 0:
        break
    for i in range((n//2)+1):
        if (check[i] and check[n-i]):
            print("{} = {} + {}".format(n, i, n-i))
            break

```

메모리 초과를 해결하기 위해 에라토스테네스의 체를 이용하여 미리 소수를 구했다.

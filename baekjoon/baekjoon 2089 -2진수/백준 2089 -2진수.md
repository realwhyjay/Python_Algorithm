# 백준 2089 : -2진수

#### Date : 21/09/9

#### Lang : Python

#### Question : [백준 2089](https://www.acmicpc.net/problem/2089)

## 최종 답안 

```python
N = int(input())
num = []

while(N != 0):
    if N % -2:
        # 파이썬에서 //연사자는 몫을 내림 연산한다.
        # 양수의 경우 상관은 없지만, 아래처럼 음수의 경우엔 문제가 생긴다.
        # -13//-2 를 해주면 몫은 6.5이다, 하지만 내림 연산을 하는 탓에 값으로 6을 출력하게 되는 것.
        # 13//2 였다면 문제가 없지만, 음수이기에 몫은 7이 나오고 나머지가 1이 나와야한다.
        # 이 때문에 리턴값에 +1을 해주어 제대로 된 몫을 구해준다.
        num.append(1)
        N = (N//-2) + 1
    else:
        num.append(0)
        N //= -2


for i in num[::-1]:
    print(i, end="")

```

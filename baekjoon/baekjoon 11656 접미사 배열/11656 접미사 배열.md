## 최종 답안 

```python
S = list(input())
temp = ""
result = []

for i in range(len(S)):
    temp = "".join(S)
    result.append(temp)
    S.pop(0)


for j in sorted(result):
    print(j)

```

까다로울 줄 알았는데 생각보다 쉬웠다.  
list로 입력을 받은 뒤, join을 사용해서 합쳐주었는데,  
다른 솔루션들을 보니 단순히 슬라이싱을 사용해서 구현을 하는 걸 봣는데  
그 편이 조금더 속도가 빠를지도...??

## 모범답안

```python
S = str(input())
S_list = []

for _ in S:
    S_list.append(S)
    S = S[1:]

for i in sorted(S_list):
    print(i)
```

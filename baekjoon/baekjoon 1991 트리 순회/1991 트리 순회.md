## 최종 답안 (클론 코딩)

구글링 해서 찾은 코드들 중 가장 잘 이해가 되는 코드로 클론 코딩했다.

```python
import sys

n = int(input())
tree = {}

for _ in range(n):
    root, left, right = sys.stdin.readline().strip().split()
    # strip()은 문자열 양끝의 공백과 '\₩'을 지워주는 역할을 하는 메소드
    tree[root] = [left, right]


def preorder(root):
    if root != '.':
        print(root, end='')  # root 값 프린트
        preorder(tree[root][0]) # 왼쪽 노드 방문
        preorder(tree[root][1]) # 오른쪽 노드 방문


def inorder(root):
    if root != '.':
        inorder(tree[root][0])
        print(root, end='')
        inorder(tree[root][1])


def postorder(root):
    if root != '.':
        postorder(tree[root][0])
        postorder(tree[root][1])
        print(root, end='')


preorder('A') # 전위 순회
print()

inorder('A') # 중위 순회
print()

postorder('A') # 후위 순회
print()

```

1. preorder traversal (전위 순회) -> **(root) (left child) (right child)**
   전위 순회는 부모노드를 먼저 방문하고 이후 자식 노드를 좌, 우 순서로 방문하는 형태이다.
   부모노드 -> 왼쪽 노드 -> 왼쪽 노드의 자식 노드(자신, 왼쪽, 오른쪽 순) -> 오른쪽 노드 -> 오른쪽 노드의 자식 노드(자신, 왼쪽, 오른쪽 순)
2. inorder traversal (중위 순회) -> **(left child) (root) (right child)**
   중위 순회는 왼쪽 자식 노드를 먼저 방문하고 부모 노드를 방문한뒤 마지막으로 오른쪽 노드를 방문한다.
   왼쪽 노드의 왼쪽 자식 노드 -> 왼쪽 노드 -> 왼쪽 노드의 오른쪽 자식 노드 -> 부모노드 -> 오른쪽 노드의 왼쪽 자식노드 -> 오른쪽 노드 -> 오른쪽 노드의 오른쪽 자식 노드
3. postorder traversal (후위 순회) -> **(left child) (right child) (root)**
   후위 순회는 왼쪽 자식 노드를 먼저 방문하고, 오른쪽 자식 노드를 전부 방문한뒤 가장 마지막으로 부모 노드를 방문한다.
   왼쪽 노드의 왼쪽 자식 노드 -> 왼쪽 노드의 오른쪽 자식 노드 -> 왼쪽 노드 -> 오른쪽 노드의 왼쪽 자식 노드-> 오른쪽 노드의 오른쪽 자식 노드 -> 오른쪽 노드 -> 부모 노드

# 백준 1978 : 소수찾기 2트

#### Date : 21/08/03

#### Lang : Python

#### Question : [백준 1978](https://www.acmicpc.net/problem/1978)

## 최종 답안 

```python
case_num = int(input())
num_list = list(map(int, input().split()))
answer = 0

if case_num == len(num_list):
    for i in num_list:
        count = 0
        for j in range(1, i+1):
            if i % j == 0:
                count += 1
        if count == 2:
            answer += 1

print(answer)
```

에라토스테네스의 체를 사용하려 했는데, 굳이 그 수들을 찾아서 저장할 필요가 없어서 일일이 체크해주었다.  
왜냐면 소수를 구하는게 아니라 소수의 갯수만 찾으면 되었기 때문!

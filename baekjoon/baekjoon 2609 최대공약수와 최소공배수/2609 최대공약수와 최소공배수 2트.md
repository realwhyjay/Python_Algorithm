## 최종 답안 

```python
num1, num2 = map(int, input().split())


def GCD(a, b):
    if a < b:
        a, b = b, a
    if b == 0:
        return a
    else:
        return GCD(b, a % b)


gcd = GCD(num1, num2)
lcm = int(num1*num2/gcd)

print(gcd)
print(lcm)

# 최소 공배수 = (두 자연수의 곱 / 최대공약수)
```

## 유클리드 호제법

두 수 중 큰 수를 작은 수로 나누고, 나머지가 생긴다면 그 나누어준 작은 수를 아까 나온 나머지로 다시 나누어준다.  
나머지가 0이될 때 까지 반복한뒤, 마지막으로 나온 나머지가 바로 최대공약수!  
최소 공배수를 구하는 식은 (두 자연수의 곱 / 최대공약수)라서 최대공약수 함수의 return 값으로 입력받은 두 수 의 곱을 나누어주었다.

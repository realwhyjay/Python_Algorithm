## 1차 답안 (시간 초과)

```python
import sys

n = int(input())
array = []

for _ in range(n):
    x = int(sys.stdin.readline())
    if x == 0:
        if len(array) == 0:
            print(0)
        else:
            print(array.pop(array.index(min(array))))

    else:
        array.append(x)
```

## 최종 답안 

```javascript
import sys
import heapq

n = int(input())
array = []

for _ in range(n):
    x = int(sys.stdin.readline())
    if x == 0:
        # 배열이 비어있는 경우에는 0을 출력한다
        if not array:
            print(0)
        # 배열이 비어있지 않은 경우에는 힙에서 가장 작은 원소를 삭제한 후 출력한다.
        else:
            print(heapq.heappop(array))
    # 입력된 값이 0이 아닐 경우에는 array에 원소를 추가한다.
    else:
        heapq.heappush(array,x)

```

## heapq 모듈

**heapq.heappush(array, x)**
첫번쨰 인자는 원소를 추가할 리스트, 두번째 인자는 추가할 원소이다. 

**heapq.heappop(array)**
인자로 원소를 삭제할 리스트를 넘기면, 해당 리스트에서 가장 작은 원소를 삭제한다.

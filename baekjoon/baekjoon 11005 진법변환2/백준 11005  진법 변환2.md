#### Question : [백준 11005](https://www.acmicpc.net/problem/11005)

## 최종 답안 

```python
num = '0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ'
n, b = map(int, input().split())
answer = ''

while (n != 0):
    answer += str(num[int(n % b)])
    n //= b

print(answer[::-1])

```

n을 b진수로 변환해주는 문제인데, 
n을 b로 나눠 몫과 나머지를 활용하여 답을 구했다.  
나머지에 해당하는 값은 미리 string 값으로 넣어둔 수 목록의 인덱스를 활용하여 변환해주었고,  
계산이 끝날때마다 n에 n을 b로 나누었을때 얻은 몫을 넣어주었다.  
이를 반복하여 값을 구하고 난 다음, 위의 경우에는 맨 끝에 대응되는 값부터 구해주었기 떄문에  
answer 값을 뒤집어서 출력해주었다.

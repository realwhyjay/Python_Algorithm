## 최종 답안 

```python
def GCD(a, b):
    if a < b:
        a, b = b, a
    if b == 0:
        return a
    else:
        return(GCD(b, a % b))


num1, num2 = map(int, input().split())
lcm = int((num1 * num2)/GCD(num1, num2))

print(lcm)

```

최소공배수를 구하는 문제.  
최대공약수를 활용하여 문제를 해결했다.  
두 수의 곱을 최대공약수로 나누어주면 최소공배수를 구할 수 있다.

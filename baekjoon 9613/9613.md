## 최종 답안

```python
# 테스트 케이스의 개수를 입력받는다
n = int(input())

# 최대공약수를 구하는 함수
def GCD(a, b):
    if b == 0:
        return a
    else:
        return GCD(b, a % b)

# 테스트 케이스 개수만큼 반복한다
for k in range(n):
    # 수를 입력받고, 반복문을 위해 갯수를 세준다.
    num_list = list(map(int, input().split()))
    num_count = len(num_list)
    sum = 0
		# 입력받은 가장 첫번째 수는 입력할 수의 갯수를 입력하기 때문에, 첫항을 제외한 나머지만 탐색하기 위해 range 0이 아닌 1부터 시작한다.
    for i in range(1, num_count-1):
        for j in range(i+1, num_count):
            sum += GCD(num_list[i], num_list[j])

    print(sum)

```

## 리스트에 map() 사용하기


map()은 리스트의 요소들을 지정된 함수로 처리해주는 함수라고한다.  
예를 들어 [1.3, 1.4, 5.1, 2.3]라는 리스트가 있다고 해보자.  
이 모든 요소들을 정수로 바꾸려면 어떻게 해야할까?  
```python
a = [1.3, 1.4, 5.1, 2.3]
for i in range(len(a)):
    a[i] = int(a[i])
```
위의 예시처럼 일일이 정수로 변환시켜줘야한다.  
하지만 map()을 사용하면 한번에  처리를 해줄 수 있다.   
```python
a = [1.3, 1.4, 5.1, 2.3]
a = list(map(int,a))
```

문제풀이에는 이러한 map을 활용하여, 입력받는 값들을 전부 정수형으로 변환하여 리스트에 넣어주었다  
```python
    num_list = list(map(int, input().split()))
```
이렇게 리스트에 정수로 정리해두었기에 문제 없이 깔끔하게 탐색이 가능했던 것이드아!
